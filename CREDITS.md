# Credits

## Icons Attribution

This project may use icons from third-party sources such as [Flaticon](https://www.flaticon.com). When icons are added, they will be listed below with proper attribution to their respective authors.

### Example Format

When adding icons, use the following format:

```markdown
- **Icon Name** (`filename.ext`) - Designed by [Author Name](author-profile-link) from [Source](source-link)
```

---

## How to Update This File

When adding new third-party icons to the project:

1. Download the icon from the source (e.g., Flaticon, FontAwesome, etc.)
2. Rename the file according to the project's naming convention
3. Add an entry to this file with proper attribution
4. Include the author's name and profile link if available
5. Update the Credits page component if needed

For detailed instructions, see [CREDITS_TEMPLATE.md](./CREDITS_TEMPLATE.md)

---

**Note:** Third-party icons may require attribution depending on their license. Always check the license terms and provide proper credit to respect the authors' work.
